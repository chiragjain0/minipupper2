Instructions

Select pupper from the menu.

Activate the controller. This is like pairing a joystick.

Activate mini pupper. This is like pressing L1 on a joystick.

Select walk from the menu.

The start and stop buttons toggle the run mode, like pressing R1 on a joystick.

When trot mode is not activated the sliders will set the values that are used by the robot when trot mode is activated.

When trot mode is active the values of a slider will be sent to the robot.
